# create-app

Generate an application form a simple json configuration. Perfect for MVP's and admin sites. Built on ideas from smalltalk and other data-model driven application creation.

## Getting started

```
1. clone repo
2. $ yarn install
3. $ yarn link
```

## Usage

```bash
$ create-app config.json
```

config.json

```json
{
  "name": "test-app",
  "template": "test"
}
```
